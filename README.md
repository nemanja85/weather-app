This APP provides current temperature in Celsius, wind speed, humidity and location coordinates. 

## Clone the project

```sh
git clone git@github.com:nemanja85/weather-app.git
```

```sh
cd weather-app
```

## Install

```sh
npm install
```

## Getting Started

```sh
npm run dev
```
